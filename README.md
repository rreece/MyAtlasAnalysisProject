MyAtlasAnalysisProject
===============================================================================

A skeleton package for developing code for analyzing xAOD data from the
ATLAS experiment at CERN.

-   author:  Ryan Reece <ryan.reece@cern.ch>
-   created: Feb 3, 2016


Getting started
-------------------------------------------------------------------------------

    source setup.sh
    rc clean
    rc find_packages
    rc compile
